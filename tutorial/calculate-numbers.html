Egison Tutorial

1. Calculate numbers

We can do arithmetic operations with '+', '-', '*', '/', and 'power'.

We can write nested expressions.

We are supporting rational numbers.

We are supporting floats, too.

We can convert a rational number to a float number with 'rtof'.

We can handle collections of numbers.
We construct collections with '{}'.

With the 'take' function, we can extract a head part of the collection.
We can construct a collection with '{}'.

We can handle infinite lists.
For example, 'nats' is an infinite list that contains all natural numbers.
Get a collection of natural numbers of any length you like.

With the 'map' function, we can operate each element of the collection at onece.

We can create a partial function using '$' as an argument.

With a 'foldl' function, we can gather together all elements of the collection using an operator you like.

Try to create a sequce of numbers '{1 1/2 1/3 1/4 ... 1/100}'.

Try to calculate '1 + 1/2 + 1/3 + 1/4 + ... + 1/100'.
Remember that you can convert a rational number to a float number with 'rtof'.

Try to calculate '1 + (1/2)^2 + (1/3)^2 + (1/4)^2 + ... + (1/100)^2'.

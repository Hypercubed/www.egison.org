Concepts of Egison

Moduralization of Pattern-matching

We can define the way of pattern-matching for each data types.

Non-linear Pattern-matching

Bindings in a pattern can be referred in the same pattern.
It is necessary to write useful patterns for pattern-matching against unfree data types.
The backtracking mechanism enables this feature.

Pattern Modularization

We can modularize useful patterns with pattern-functions.
Since a pattern-function has lexical scoping, we can reuse useful patterns in many places without worry of name clash.
